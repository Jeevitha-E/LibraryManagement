# LibraryManagement

A REST API built using Spring Boot, Hibernate, JDBC, and PostgreSQL to manage library book records.

## Features
- Add a new book
- View all books
- View a book by ID
- Update book details
- Delete a book

## Technologies Used
- Java 17
- Spring Boot 3.2
- Hibernate (JPA)
- JDBC
- PostgreSQL
- Postman (for testing)
